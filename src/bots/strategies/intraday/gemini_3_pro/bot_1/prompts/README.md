# Prompts para Estrategia INTRADAY

Esta carpeta contendrá los prompts específicos para la estrategia INTRADAY con Gemini 3 Pro.

## Estructura de Prompts

Los prompts se organizarán según el tipo de análisis:

### 1. System Prompt
Define el rol y contexto del agente IA para la estrategia INTRADAY.

### 2. User Prompt
Contiene los datos de mercado y la pregunta específica para análisis.

### 3. Prompts Especializados
- **Identificación de niveles**: Soporte, resistencia, pivotes
- **Análisis de tendencia**: Tendencia intraday en múltiples timeframes
- **Señales de entrada**: Condiciones para LONG/SHORT
- **Gestión de posiciones**: Reevaluación y ajuste de stops

## Configuración de Gemini 3 Pro

Los prompts están diseñados para aprovechar:
- **thinking_level: HIGH** - Razonamiento profundo para análisis técnico
- **code_execution** - Cálculos matemáticos precisos de indicadores
- **media_resolution: high** - Análisis de alta calidad
